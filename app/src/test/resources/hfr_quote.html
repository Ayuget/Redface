Bonjour,
 
Je vous présente Redface, le nouveau client HFR pour Android, qui se veut dans la continuité du superbe travail de Toyonos sur HFR4droid.
 
J'ai voulu concevoir un client qui propose une interface au goût du jour, et surtout, adaptée aussi bien aux téléphones qu'aux tablettes. Considérez cette première version comme une version bêta car beaucoup de choses manquent, mais la base est là !

Changelog des dernières versions :
 
Version 2.0.0 :
 
Attention, voilà les messages privés et le multi-quote ! Et Redface change de tête au passage, avec une nouvelle icône :O
 
Grosse mise à jour, puisqu'elle contient :
 

  • [FEATURE] La gestion des messages privés
  • [FEATURE] Le multi-quote
  • [FEATURE] Mode d'affichage compacte de la liste des sujets
  • [FEATURE] Gestion de la taille de la police dans les sujets
  • [FEATURE] Suppression d'un message
  • [FEATURE] Ajout d'un message en favori
  • [FEATURE] Indicateur d'édition du message
  • [FEATURE] Partage/copie d'un lien vers un sujet
  • [FEATURE] Ajout d'un bouton pour ajouter une image dans la fenêtre de réponse
  • [BUGFIX] Gestion complète des liens internes
  • [BUGFIX] Ajout de smileys lorsqu'on édite un message
  • [BUGFIX] Problème de rafraichissement de la page après un post
  • [BUGFIX] Memory leaks


Version 2.1.0 :
 
Petite release apportant quelques nouvelles fonctionnalités
 

  • Correction des crashs les plus fréquents
  • Ajout d'un navigateur interne (activé par défaut mais désactivable dans les options)
  • Vous pouvez désormais sauvegarder et partager des images, ainsi que voir leurs données EXIF
  • Correction de la recherche de smileys


Version 2.1.1 :
 
Release de bugfixes et petites features, en attendant l'intégration de reho.st dans la prochaine version !
 

  • [FEATURE] Les favoris/drapeaux peuvent maintenant être supprimés
  • [FEATURE] EgoQuote : mise en évidence des messages dans lesquels vous êtes cité
  • [FEATURE] Mise en évidence des messages de la modération
  • [BUGFIX] Correction de la taille des smileys lorsqu'on poste un message
  • [BUGFIX] Tous les liens vers HFR sont désormais correctement ouverts
  • [BUGFIX] Correction de lien internes non fonctionnels
  • [BUGFIX] Multi-quote désormais possible sur plusieurs pages
  • Autres bugfixes divers


Version 2.3.0 :
 
Fonctionnalités apportées par gic-gic, un énorme merci à lui !
 

  • FEATURE : Blacklist : ignorer les messages de certains forumeurs
  • FEATURE : Option pour rafraîchir la liste des sujets lors du retour sur celle-ci
  • FEATURE : Options d'affichage pour les images/avatars/smileys (Toujours, si WiFi, ...)
  • FEATURE : Boutons "barré" et "souligné" lors de la rédaction d'un post
  • FEATURE : Option "effacer le drapeau" depuis le menu contextuel
  • BUGFIX : EgoQuote dans les messages contenant un lien
  • BUGFIX : récupération de l'avatar
